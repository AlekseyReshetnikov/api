HeadHunter API

По-русски | In english

HeadHunter API is a free toolset for integration of HeadHunter with your product.

First of all, read the general information on work with API and the requirements for logos and application names use.

To use services that require user authorization, you should register the application at https://dev.hh.ru and set up the authorization process.
